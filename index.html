Zyrouge-API

Quick Zyrouge API wrapper for beginners.

Features

  • Beginner friendly

  • Easy to use

  • Asynchronous

Quick Example


const { API } =  require("new-zyrouge-api");

const  api  =  new  API("Super Secret API Token");

  

api.catfact().then(console.log).catch(e => console.log(e));
